This project is a sample implementation of a stock holdings screen built with MVVM, Clean Architecture, Kotlin Coroutines, Flow, Retrofit, and Room.
It showcases a modern Android application structure with clear separation of concerns, offline support through local caching, and comprehensive unit testing.

Tech Stack:
Kotlin, MVVM + Clean Architecture, Coroutines + Flow, Retrofit, Room Database, ViewBinding, JUnit, Coroutines Test
